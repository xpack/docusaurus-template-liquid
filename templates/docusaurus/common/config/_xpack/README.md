# website/config

Configuration files for various tools.

- `api-extractor-original` - the result of `api-extractor --init`
- `api-extractor.json` - the useful configuration, without all comments
- `tsconfig-api-extractor.json` - the Typescript configuration to compile the sources for api-extractor
- `tsdoc2docusaurus.json` - the configuration for the Docusaurus converter
