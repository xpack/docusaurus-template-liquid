### Start the local web server

Execute the npm script **clear** followed by **start** in the website sub-project,
or execute the following within the project folder:

```sh
npm run clear -C ~/Work/{{githubProjectOrganization}}/{{githubProjectName}}.git/website
npm run start -C ~/Work/{{githubProjectOrganization}}/{{githubProjectName}}.git/website
```

Navigate to the **Maintainer Info** page,
the **[Start the local web server](http://localhost:3000{{baseUrl}}docs/maintainer/#start-the-local-web-server)** section.
